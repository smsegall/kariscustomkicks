Process

Each pair of shoes are:

  1. Thoroughly cleaned and prepped before painting

  2. Painted using the best leather paints or dyes depending on the shoe material

  3. Glazed in a finishing coat that protects the paint or dye

  4. Laced up using the original or optional premium laces
  5. Shipped out by USPS priority mail
